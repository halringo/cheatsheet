Vim Cheat Sheet

NERD tree

:NERDTree <Space>nn

新しい NERD tree を開く。

ツリーのルートは、指定された引数に依存し、3つのケースがある。 引数を指定しない場合は、現在のディレクトリが使用される。 ディレクトリが指定された場合、それが使用される。 ブックマーク名が指定された場合、対応するディレクトリが使用される。

:NERDTreeToggle <Space>nt

NERD Tree がすでに現在のタブに存在する場合は、それを 再度開き、再レンダリングする。 閉じる。 現在のタブに NERD Tree が存在しない場合は、:NERDTreeコマンドと同じ動作をする。

:NERDTreeMirror <Space>nm

現在のタブに、別タブの既存の NERD tree を共有する。 それらは実際には同じバッファなので、片方のツリーに加えられた変更は、両方に反映される。

NERD tree が1つしか存在しない場合、そのツリーが自動的にミラーリングされる。 複数存在する場合、スクリプトはミラー先のツリーを聞いてくる。

:NERDTreeClose <Space>nw

現在のタブの NERD Tree を閉じる。

:NERDTreeFind <Space>ni

ツリー内で現在のファイルを検索します。

ツリーが存在せず、現在のファイルが Vim の CWD 下にある場合は、CWD でツリーを初期化し、ファイルを表示します。 それ以外の場合は、現在のファイルのディレクトリでツリーを初期化する。

いずれの場合も、現在のファイルが表示され、カーソルがそのファイルの上に置かれる。

:NERDTreeCWD <Space>nc

現在のディレクトリにツリーのルートを変更する。 NERD tree が現在のタブに存在しない場合、新しいツリーが開かれる。

:NERDTreeFocus <Space>nf

NERD tree にカーソルを移動させる。NERD tree が開いていない場合は、開いてからカーソルを移動させる。

:NERDTreeFocusは、ヘルプには書かれていない。
